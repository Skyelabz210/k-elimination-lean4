The Core Insight

The entire proof rests on a single key congruence:

key_congruence (Lean 4)
theorem key_congruence (X M A : N) : X % A = (X % M + (X / M) * M) % A := by have h : X = X % M + (X / M) * M := div_mod_identity X M calc X % A = (X % M + (X / M) * M) % A := by rw [<- h]

This proves: va = vm + k*M (mod A)
Multiply by M-1 to recover k exactly.
